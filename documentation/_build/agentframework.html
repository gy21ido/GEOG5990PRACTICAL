agentframework module¶

This file contains all codes about the agents and agents predator (Wolves). Here, behaviours such as how they move, share with their neighbours, eat, and breed to populate themselves are included. Properties of each agent is also included in the file.

eat()¶

This makes the agents eat the environment and their eating is controlled. If an agent eats at least 100 units, it vomits and empties its bowel (Store) and stops eating.

Increased store value after eating.

>>> print (agent)
    Location: (15, 148) Store: 0.0
>>> agent.eat()
>>> print (agent)
    Location: (15, 148) Store: 10.0
