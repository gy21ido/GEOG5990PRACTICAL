Model module¶

This model shows an environment with full breeding population of agents and wolves. Agents interact with the environment by eating it, while wolves interact with agents by eating them. Breeding is added to populate the environment, as both wolves and agents breed with one another.

Model.gen_function(b=[0])¶

Function to generate the frame

bint, optional

DESCRIPTION. The default is [0].

int

value for the frames generated.

Model.run()¶

Runs the animated model

Animation is generated from matplotlib function that calls the gen_function

Model.update(frame_number)¶

Updates the tkinter window/ frame

frame_number : int (optional)

A change in the plot within tkinter.

>>> update(0)
